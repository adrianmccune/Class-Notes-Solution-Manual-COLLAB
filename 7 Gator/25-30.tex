% type problem below this line
Follow the instructions below.

Write $(b^2)^3$ without exponents.
$$(b^2)^3=\underline{\qquad}$$
\vspace{.2 cm}

Fill in the blank.
$$(b^2)^3=b^{\hspace{0.1cm}\underline{\phantom{AA}}}$$





%type problem above this line
\ifbool{answerKey}%
{
\vfill \textbf{Answer:}
%type answer below this line

$(b^2)^3 = b\cdot b\cdot b\cdot b\cdot b\cdot b$

$(b^2)^3 = b^6$

}%
{}
