Evaluate the following.

$$(16 - 14.6)^2 + 69.5\div 0.5$$





%type problem above this line
\ifbool{answerKey}%
{\begin {center}
Step 1: Identify operations and elements.
$$Parentheses, exponent, addition and division$$
Step 2: Determine order using PEMDAS.
$$Parentheses, exponent, addition, then division$$
Step 3: Solve the inside of parentheses by subtraction.
$$16 - 14.6 = 1.4$$
so we have, 
$$(1.4)^2 + 69.5\div 0.5$$
Step 4: Evaluate the exponent term.
$$(1.4)^2 = 1.96
1.96 + 69.5\div 0.5$$
Step 5: Divide 69.5 by 0.5.
$$69.5\div 0.5 = 139$$
Step 6: Add.
$$1.96 + 139 = 140.96$$
\end {center}}

% type problem below this line
Evaluate the following.

$$66 \frac{1}{4} - 5.84$$

Write your answer as a decimal.





%type problem above this line
\ifbool{answerKey}%
{\begin {center}
Step 1: Convert $66 \frac{1}{4}$ to a decimal number starting with the fractional part.
$$\frac{1}{4} is 1 divided by 4.
1\div 4 = 0.25$$
Step 2: Add the fractional part to the whole number.
$$66 + 0.25 = 66.25$$
Step 3: Rewrite.
$$66.25 - 5.84$$
Step 4: Subtract.
$$66.25 - 5.84 = 60.41$$
\end {center}}
\vfill \textbf{Answer:}

$60.41$
