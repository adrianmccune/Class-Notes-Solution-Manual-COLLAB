Evaluate the following.

$$(16 - 14.6)^2 + 69.5\div 0.5$$





%type problem above this line
\ifbool{answerKey}%
{\textbf{Explanation:}
\begin {itemize}
\item Identify operations and elements.
$$Parentheses, exponent, addition and division$$
\item Determine order using PEMDAS.
$$Parentheses, exponent, addition, then division$$
\item Solve the inside of parentheses by subtraction.
$$16 - 14.6 = 1.4$$
so we have, 
$$(1.4)^2 + 69.5\div 0.5$$
\item Evaluate the exponent term.
$$(1.4)^2 = 1.96
1.96 + 69.5\div 0.5$$
\item Divide 69.5 by 0.5.
$$69.5\div 0.5 = 139$$
\item Add.
$$1.96 + 139 = 140.96$$
\end {itemize}}
\vfill \textbf{Answer:}

% type problem below this line
Evaluate the following.

$$66 \frac{1}{4} - 5.84$$

Write your answer as a decimal.





%type problem above this line
\ifbool{answerKey}%
{\textbf{Explanation:}
\begin {itemize}
\item Convert $66 \frac{1}{4}$ to a decimal number starting with the fractional part.
$$\frac{1}{4} is 1 divided by 4.
1\div 4 = 0.25$$
\item Add the fractional part to the whole number.
$$66 + 0.25 = 66.25$$
\item Rewrite.
$$66.25 - 5.84$$
\item Subtract.
$$66.25 - 5.84 = 60.41$$
\end {itemize}}
\vfill \textbf{Answer:}
%type answer below this line
$60.41$
