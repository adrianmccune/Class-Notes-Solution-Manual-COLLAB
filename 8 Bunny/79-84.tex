Suppose that the functions $f$ and $g$ are defined as follows:
$$f(x) = 4x - 5$$
$$g(x) = \sqrt{3x+1}$$
Find $\frac{f}{g}$ and $f-g$. Then, give their domains using interval notation.


%type problem above this line

{
\vfill \textbf{Answer:}
%type answer below this line

$\left(f-g\right) = 4x-5-\sqrt{3x+1}$

Domain of $\frac{f}{g}: (-\frac{1}{3}, \infty)$

Domain of $f-g : \left[-\frac{1}{3}, \infty \right)$
