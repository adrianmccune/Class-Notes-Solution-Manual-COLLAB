% 88. type problem below this line

The one-to-one function $g$ is defined below.
$$g(x) = \frac{8x-3}{5x+8}$$
Find $g^{-1}(x)$, where $g^{-1}$ is the inverse of $g$. 

Also state the domain and range of $g^{-1}$ in interval notation.


%type problem above this line
\ifbool{answerKey}%
{
\vfill \textbf{Answer:}
%type answer below this line

$g^{-1}(x) = \frac{-8x-3}{5x-8}$
\\

Domain of $g^{-1}: \left( -\infty, \frac{8}{5} \right) \cup \left( \frac{8}{5}, \infty \right)$
\\

Range of $g^{-1}: \left( -\infty, -\frac{8}{5} \right) \cup \left( -\frac{8}{5}, \infty \right)$

}%
{}
