\vspace{1 cm}

\noindent This module focuses on more complete order of operations, the connection between fractions, decimals, percentages, and application problems involving percentages. Concepts in \textbf{Module B} are supported by \textbf{Lessons 5A - 5D}, which help to make the connection between fractions, decimals, and percentages. 

\vspace{1 cm}
{\large Objectives:}

\vspace{.3 cm}
By the end of this module, students will be able to:
\begin{itemize}
    \item Apply Order of Operations with problems including fractions. 
    \item Convert values between decimals, fractions, and percentages. 
    \item Use percentage to find tax, sales prices, and total amounts in real-world problems. 
    \item Identify segments within a circle and calculate circumference and area. 
    \item Complete problems using addition and subtraction of integer values. 
\end{itemize}
